# MERNG-simple-blog-boilerplate
## Main Features: 
- [x] Simple Blog boilerplate.
- [x] Simple Admin/User system.
- [x] Guest can register new accounts.
- [x] React Context Api.
- [x] Semantic UI.
- [x] Dynamic Navigation bar.
- [x] User profile page.
- [x] Blog post Comments.
